# 🛒 Shopping Management System (C++)

## 📌 Project Overview

The **Shopping Management System** is a console-based application developed in **C++** that simulates a simple shopping environment. It allows users to view products, purchase items, generate bills, and manage shopping records efficiently.

This project demonstrates core programming concepts such as file handling, object-oriented programming, and data management using C++.

---

## 🎯 Objectives

* Implement a basic shopping system using C++
* Understand file handling for storing records
* Apply Object-Oriented Programming (OOP) concepts
* Simulate real-world shopping and billing system

---

## 🚀 Features

### 👤 User Features

* 📋 View available products
* 🛒 Select and purchase items
* 💰 Automatic bill generation
* 📊 View total cost of purchases

### 🛠️ Admin Features

* ➕ Add new products
* ✏️ Update product details
* ❌ Delete products
* 📦 Manage product inventory

---

## ⚙️ Tech Stack

* 💻 **Language:** C++
* 🗄️ **File Handling:** Text/Binary Files
* 🧠 **Concepts Used:** OOP, Functions, Loops, Arrays
